# ⏰ Python Countdown Timer

A versatile, beginner-friendly countdown timer built with Python. Features multiple versions from simple terminal-based timers to a full GUI application with presets and progress bars!

## 📋 About The Project

This Countdown Timer project is designed to help Python beginners learn programming concepts through a practical, useful application. Starting from a basic 10-line script to a feature-rich GUI application, this project demonstrates how to build increasingly complex programs while maintaining functionality.

### ✨ Features

**All Versions Include:**
- 🎯 Accurate countdown timing
- ⌨️ User-friendly interface
- ⏸️ Stop/Interrupt handling
- 🔊 Audio alerts when timer completes

**Enhanced Version Features:**
- 📊 Visual progress bar
- 🎨 Preset time options (1, 5, 10, 20, 25, 30 minutes)
- 📱 Hours:Minutes:Seconds display
- 🧹 Auto-screen clearing
- 🔄 Multiple timer sessions without restarting

## 🚀 Getting Started

### Prerequisites

- **Python 3.x** installed on your system
- No external libraries required for terminal versions
- **tkinter** (comes with Python) for GUI version

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mobrahi/countdown_timer.git
   cd countdown_timer
   ```

2. **Run any version you want**
   ```bash
   # Simple version
   countdown_timer.py
   
   # Enhanced version
   timer_with_multi_features.py

```

## 🎮 How to Use

### Version 1: Simple Timer
```bash
⏰ SIMPLE COUNTDOWN TIMER ⏰
==============================
Enter number of seconds to countdown: 10

Time left: 10 seconds
Time left: 9 seconds
...
🎉 TIME'S UP! 🎉
```


### Version 2: Enhanced Timer
```bash
    ⏰ COUNTDOWN TIMER ⏰
========================================

Options:
1. ⏱️  Set custom time
2. 📝 Use preset times
3. ❌ Exit
```
